Sobre a Oceano Games

A Oceano Games cria uma ponte entre o ritmo do oceano, a fluidez do som e preocupação com jogabilidade. Acreditamos em criar experiências onde o som é o elemento central que conduz o jogo.

Na Oceano Games, celebramos inclusão, criatividade e aventura.

Nossos Jogos em Desenvolvimento

The Hitmaker

The Hitmaker coloca você no papel de um produtor musical, gerenciando um estúdio de gravação. O jogador deve contratar músicos, coordenar sessões de gravação e criar faixas únicas. Cada decisão influencia o resultado final das músicas e o sucesso do estúdio.

Patches

Inspirado por sintetizadores modulares, Patches é um puzzle onde o jogador deve conectar circuitos de som para criar padrões musicais. O objetivo é manipular ondas sonoras e sinais para resolver desafios e harmonizar a trilha sonora do jogo.

Runegarden

Em Runegarden, os jogadores encontram um puzzle minimalista, zen e relaxante, onde o som é a chave para solucionar cada fase. O jogador coloca runas em um grid, criando harmonia para avançar no jogo.

Marble Machines

Inspirado pela famosa Marble Machine, este jogo combina elementos de física e música. O jogador alinha peças em um grid, e cada peça representa um trecho musical, criando uma experiência interativa de lógica e ritmo musical.

The Jazzclub

Um crime aconteceu em um clube de jazz. Em The Jazzclub, o jogador investiga usando sons para encontrar pistas. A exploração sonora é a chave para resolver o mistério nesse jogo de aventura em 2D com clima noir.

Mix n Match

Mix n' Match é um puzzle musical onde o jogador alinha instrumentos em um grid para criar a mixagem perfeita. Usando mecânicas de match e merge, o jogador deve harmonizar as faixas musicais.

Gearbox

Em Gearbox, o jogador manipula engrenagens e peças em um grid para reproduzir o som correto. O desafio é montar corretamente o mecanismo musical para que ele toque perfeitamente.

The Cube

Inspirado no cubo mágico e no jogo Genius, The Cube é um puzzle sonoro onde o jogador gira as peças para recriar a melodia correta, usando uma mecânica retrô dos anos 80.

Patches: Synthflow

Uma versão alternativa de Patches, com foco no fluxo de sinais sonoros. Em Synthflow, o jogador conecta patch cables para explorar combinações de ondas e efeitos sonoros, criando uma experiência técnica e musical profunda.
